shadiaoMusic

A android app to create music for fun.


### set up 
```
npm run install
npm run android
```


#### 颜色值对应

红：'#d93732'
黄：'#f2af22'
紫：'#985978'
淡黄：'#fdf2dc'
绿：'#a7b11c'


#### 项目目录结构

src
    audio //音频提取
        audio.js //音频主界面
    video //视频提取
        camera.js //相机组件
        video.js //视频主界面
    record //录音
        record.js //录音组件
    piano //钢琴
        Piano.js  //钢琴组件
        Key.js   //钢琴琴键
    genMusic
        genMusic.js  //传入数组生成音乐
    img    //存放图片资源
